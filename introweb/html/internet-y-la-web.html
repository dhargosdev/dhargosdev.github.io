Internet y la Web

Internet no es lo mismo que la Web

Internet es la red mundial que conecta dispositivos y servidores. La Web es un servicio que funciona sobre Internet y que nos permite consultar páginas enlazadas entre sí.

Una forma sencilla de entenderlo:

Concepto Qué es Ejemplos
Internet Infraestructura de comunicación Redes, routers, servidores, cables, Wi-Fi, fibra óptica.
Web Servicio que usa Internet Sitios web, páginas web, enlaces, navegadores.

Usamos Internet para muchas cosas: correo electrónico, mensajería, videollamadas, juegos online y también para navegar por la Web.
