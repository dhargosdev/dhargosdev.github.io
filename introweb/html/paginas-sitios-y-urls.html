Páginas, sitios y URLs

Texto

Una página web es un recurso que puede mostrarse en un navegador. Puede contener texto, imágenes, vídeos, enlaces, formularios y otros elementos.

Un sitio web es un conjunto organizado de páginas relacionadas. Por ejemplo, la web de un instituto puede tener una página de inicio, una página de noticias, una página de contacto y varias páginas para departamentos.

Una URL es la dirección que usamos para localizar un recurso en Internet.

Ejemplo:

https://www.ejemplo.com/recursos/tema1.html

Partes básicas:

  • https:// : protocolo seguro.
  • www.ejemplo.com : dominio.
  • /recursos/tema1.html : ruta del recurso.
