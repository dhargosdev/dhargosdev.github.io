Navegadores y servidores

¿Qué ocurre cuando abrimos una web?

Imagina que Ana escribe en el navegador la dirección de la web de su instituto.

  1. El navegador interpreta la URL.
  2. Busca dónde está el servidor  correspondiente.
  3. Solicita la página.
  4. El servidor responde enviando los recursos necesarios.
  5. El navegador interpreta esos recursos y muestra la página.

En este proceso, el navegador actúa como cliente y el servidor entrega la información solicitada.
