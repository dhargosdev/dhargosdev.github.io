Servidores, hosting y centros de datos

Texto

Las páginas web y sus datos se alojan en servidores. Un servidor puede ser un ordenador físico, una máquina virtual o un servicio en la nube.

Muchos servidores están en centros de datos: edificios preparados para mantener equipos funcionando todo el tiempo, con buena conexión, seguridad, refrigeración y sistemas eléctricos de respaldo.

Cuando contratamos un lugar para guardar y publicar una web hablamos de hosting o alojamiento web.

Nota importante: una web moderna no siempre está formada por archivos simples guardados en una carpeta. También puede usar bases de datos, servicios externos, almacenamiento distribuido o redes CDN.
