HTML: la base de la Web

HTML estructura el contenido

HTML es el lenguaje de marcado básico de la Web. Sirve para indicar la estructura del contenido: títulos, párrafos, enlaces, listas, imágenes, tablas, formularios, etc.

Ejemplo:

<!doctype html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>Mi primera página</title>
</head>
<body>
  <h1>Mi primera página web</h1>
  <p>Hola, mundo.</p>
  <p><a href="otra-pagina.html">Ir a otra página</a></p>
</body>
</html>

HTML indica qué es cada parte del contenido. El navegador interpreta ese código y muestra la página. Para controlar el aspecto visual se suele usar CSS, y para añadir interactividad se usa JavaScript.
