Introducción a las páginas web

Todos los días usamos páginas web para estudiar, buscar información, comunicarnos, comprar, jugar o consultar servicios. Pero, ¿qué es exactamente una página web? ¿Dónde está almacenada? ¿Qué hace el navegador cuando escribimos una dirección? ¿Qué papel tiene HTML?

En esta unidad vamos a descubrir cómo funciona la Web desde dentro, sin entrar todavía en la creación práctica de una página. Primero entenderemos los conceptos; después, en otra unidad, podremos empezar a construir nuestras propias páginas.
